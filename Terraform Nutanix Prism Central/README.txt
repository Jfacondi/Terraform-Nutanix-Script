How to use:
1) Download terraform from https://www.terraform.io/downloads.html
2) Unzip the file and add the terraform binary to your PATH // on linux you can do this by running `sudo mv terraform /usr/local/bin/`
3) fill in the terraform.tfvars file with your credentials and specs for the vms
4) open terminal and run `terraform init`
5) run `terraform plan` to see what terraform will do
6) run `terraform apply` to create the vms
7) run `terraform destroy` to destroy the vms
